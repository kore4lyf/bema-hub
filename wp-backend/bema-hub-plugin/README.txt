WordPress plugin for Bema Hub platform integration.

== Description ==

Bema Hub Plugin provides integration with the Bema Hub platform, offering enhanced functionality for managing and tracking user interactions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bema-hub-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
